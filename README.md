<h3>About</h3>
<p>Allows users to create maps for destinations they want to go to and add places to visit while traveling to each destination</p>

<h3>API's used</h3>
<ul>
  <li>Angular Google Maps.</li>
  <li>Google Places with Autocomplete.</li>
</ul>
